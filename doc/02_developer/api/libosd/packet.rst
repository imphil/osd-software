osd_packet class
----------------

Representation of a single packet in the debug interconnect.

Accessor functions encapsulate reading and writing fields inside the packet.

Usage
^^^^^

.. code-block:: c
 
  #include <osd/osd.h>
  #include <osd/packet.h>
